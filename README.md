# The Machina Series
This repo contains the different CPU/VM emulators of the Machina family. All of which, are meant for educational and experimental purposes.

| Name   | Word Length | Purpose                              |
| ------ | ----------- | ------------------------------------ |
| Argon  | 32-bit      | Base/Reference implementation        |
| Helium | 32-bit      | Adds floating-point, `mul` and `div` |
